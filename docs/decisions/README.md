# Registros de Decisões Arquiteturais (ADRs)

Documentos curtos que capturam decisões significativas: o contexto, a escolha feita e suas consequências. Um arquivo por decisão, numerado e imutável após aceito (substitua em vez de reescrever).

Formato: `ADR-NNNN-slug-curto.md` com as seções **Status**, **Contexto**, **Decisão** e **Consequências**.
