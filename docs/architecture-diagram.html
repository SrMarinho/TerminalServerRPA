TerminalServerRPA — Arquitetura

Automação RPA para ERP legado via Terminal Server

Interface

  • • FastAPI localhost (127.0.0.1)
  • • Tailwind SPA + WebSocket ao vivo
  • • CLI Typer (vault, run, logs)
  • • Token Bearer automático

Engine

  • • TaskPool + TaskRunner assíncrono
  • • Pausa/Retoma/Cancela/Skip
  • • Execução persistida em SQLite
  • • Auto-descoberta de tasks por filesystem

Infraestrutura

  • • Vault: keyring + Fernet
  • • Logger: structlog (JSON + console + WS)
  • • Instância única: mutex + pipe
  • • Auto-update: GitHub Releases

TerminalServerRPA • Python 3.14 • FastAPI • Playwright
